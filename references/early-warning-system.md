# Early Warning System (先知预警体系)

基于孙子"庙算"思想，在病毒式传播高峰前识别机会。

---

## 核心理念

> "夫未战而庙算胜者，得算多也。"
>
> 情报价值 = 准确度 × 时效性。发现热点时已是峰值，等于没发现。

---

## 五事前兆指标体系

### 1. 道之萌 (User Pain Emerging)

**定义**：用户对现有产品的不满开始聚集

| 指标 | 监测方式 | 预警阈值 | 数据源 |
|------|----------|----------|--------|
| 抱怨帖数量 | 关键词监控 | 周增 >50 条 | Reddit, Discord |
| 负面评论比例 | 情感分析 | >30% 负面 | Steam, G2 评论 |
| "alternative to X" 搜索 | Trends 监控 | 周增 >100% | Google Trends |
| 竞品讨论热度 | 社区扫描 | 新竞品被提及 >20次/周 | Reddit |

**Sprunki 案例回溯**：
- 2024年8月前，Incredibox 用户抱怨"内容太少"、"想要更多角色"
- 如果当时监控到这些信号，可提前1个月预判 mod 需求

---

### 2. 天之变 (Timing Shift)

**定义**：外部环境变化创造机会窗口

| 指标 | 监测方式 | 预警阈值 | 数据源 |
|------|----------|----------|--------|
| Google Trends 7日斜率 | 自动计算 | >200% 周增 | Trends API |
| 社交媒体提及量 | 爬虫/API | 日增 >500% | Twitter, TikTok |
| 新闻/媒体覆盖 | 搜索监控 | 主流媒体首次报道 | Google News |
| 季节性因素 | 日历标记 | 假期前2周 | 内部日历 |

**Sprunki 案例回溯**：
- 2024年9月 Trends 斜率突破 500%
- 如果设置 200% 阈值，可提前2周触发预警

---

### 3. 地之隙 (Competition Gap)

**定义**：市场存在可占领的空白地带

| 指标 | 监测方式 | 预警阈值 | 数据源 |
|------|----------|----------|--------|
| SERP 空位率 | 手动/半自动审计 | >40% 无专业站 | Google SERP |
| 域名可用性 | 域名检查 | 核心域名可注册 | Namecheap API |
| YouTube 视频质量 | 抽样评估 | >50% 为低质量 | YouTube |
| 权威站点覆盖 | 搜索检查 | 无 IGN/PCGamer 等报道 | 手动 |

**Sprunki 案例回溯**：
- 2024年9月，sprunki.com/sprunki.org 均未被注册
- SERP 全是 Scratch 页面，无专业游戏站
- 如果当时检测，地之隙评分应为 9/10

---

### 4. 将之动 (Creator Movement)

**定义**：内容创作者开始涌入

| 指标 | 监测方式 | 预警阈值 | 数据源 |
|------|----------|----------|--------|
| 新视频发布量 | YouTube API | 日增 >100 视频 | YouTube |
| 新 mod/remix 数量 | 平台监控 | 日增 >10 个 | Scratch, itch.io |
| 创作者讨论 | 社区监控 | "should I make X content" >5帖 | Reddit |
| 直播覆盖 | Twitch API | 新主播 >20 人 | Twitch |

**Sprunki 案例回溯**：
- 2024年10月，YouTube 日增视频量突破 500
- Phase 3 发布后，mod 创作者日增 >50
- 这是最强的"爆发确认"信号

---

### 5. 法之成 (Product Maturity)

**定义**：产品本身具备传播潜力

| 指标 | 监测方式 | 预警阈值 | 数据源 |
|------|----------|----------|--------|
| 可玩性 | 人工测试 | 5分钟内上手 | 人工 |
| 特色差异 | 人工评估 | 有1个明确卖点 | 人工 |
| 社交属性 | 功能检查 | 可分享/可互动 | 人工 |
| 视觉吸引力 | 截图评估 | 截图有传播性 | 人工 |

**Sprunki 案例回溯**：
- 拖拽创作 = 5分钟上手 ✓
- 恐怖元素 = 明确差异化卖点 ✓
- 可分享作品 = 社交属性 ✓
- 角色设计 = 视觉吸引力 ✓
- 法之成评分：9/10

## 超速预警模式 (60秒判定)

当任一信号触发阈值时，自动执行：

```
第 0 秒：触发信号检测
第 10 秒：抓取 Google Trends 7日数据
第 20 秒：检索目标平台 24h 热度
第 30 秒：检查 SERP 前10结果
第 40 秒：计算五事得分 (已知项)
第 50 秒：生成判定结果
第 60 秒：推送通知

输出格式：
┌────────────────────────────────────┐
│ ⚡ 超速预警：{keyword}              │
│ 五事得分：{X}/5                    │
│ 预警等级：{颜色}                   │
│ 建议行动：{GO/WAIT/SKIP}           │
│ 详情链接：{URL}                    │
└────────────────────────────────────┘
```
